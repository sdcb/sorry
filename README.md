# FFmpeg生成GIF动图

### 体验网站：
* 新版本（基于Gradio.NET）：https://sorry.starworks.cc:88/
* 老版本：https://ffmpeg-sorry-demo.starworks.cc:88/

### 要点：
* 视频解码
* 将每一帧转换为BGRA像素格式
* 使用Direct2D读取并绘制字幕
* 将每一帧输入视频过滤器，转换为PAL8格式
* 将PAL8编码像素格式的帧编码为gif
